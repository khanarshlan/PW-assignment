# some examples and explanation of operators category ?  

- Arithmetic Operators: +, -, *, /, % (modulus)

```javascript example
let a = 5;
let b = 6;
console.log( a + b )
console.log( a - b )
console.log( a * b )
```

- Assignment Operators: =, -=, +=, etc

```javascript example
let a = 5;
let b = a += 6;
console.log(a)
```

- Comparison Operators: ==, <, >, etc

```javascript example
let a = 5;
let b = 6;

if (a < b){
    console.log("a is small")
}
```
- Logical Operators: &&, ||, etc

```javascript example
let a = 5;
let b = 6;

if (a = b || a < b){
    console.log("a is small")
}
```
- Increment and Decrement Operators: ++, --

```javascript example
let a = 5;
let b = ++a;

if (a = b || a < b){
    console.log(b)
}
// used prefix
```
